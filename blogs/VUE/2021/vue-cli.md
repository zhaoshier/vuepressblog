### 1.安装node.js，全局安装npm，并配置淘宝镜像

### 2.命令行安装全局vue-cli
```
npm install --global @vue/cli-init
```

安装成功后如图所示：
![](https://gitee.com/zhaoshier/blogimage/raw/master/images/init项目.png)

### 3.创建项目
```
vue init webpack 项目名
```

并选择对应配置，如下图：
![](https://gitee.com/zhaoshier/blogimage/raw/master/images/init项目.png)

### 4.创建成功后，启动项目   
```
npm run dev
```
